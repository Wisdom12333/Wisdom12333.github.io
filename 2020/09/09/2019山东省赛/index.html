B_Flipping Game

DP.
n个灯,有0态和1态,每次翻转m个,进行k次.问从始态到末态有多少种可能.
转自 https://www.cnblogs.com/CSGOBESTGAMEEVER/p/10900928.html;
第i位灯泡始态与末态相同,翻转偶数次,不同则翻转奇数次;
假设共有 $ Odd $ 灯泡要翻转奇数次;
DP[ i ][ j ],表示 i 次后有 j 个灯泡翻转了奇数次,那么 i + 1 次时,从 j 中取出 l 个翻转,则奇数次为 j - l ,从剩下被翻转偶数次中取出 m - l 翻转,奇数次为 j - l + m - l .DP[ i + 1 ][ j - l + m - l ];
那么使用排列组合,得到状态转移方程.
DP [ i + 1 ][ j - l + m - l ] = DP [ i + 1 ][ j - l + m - l ] $ C_j^l $ $ C_{n-j}^{m-l} $;
假设初态为 DP [ 0 ][ Odd ] = 1;输出为 DP [ k ][ 0 ];
同时注意对 998244353 取模;

C_Wandering Robot

有一个机器人根据基于U D R L四个指令的指令串进行上下左右移动,自原点开始,重复k次,每个指令移动一个单位.求对于原点的绝对距离(| $ x $ | + | $ y $ |);
一开始想到的是计算k-1次指令后的位置,循环最后一次更新最大距离,后来测试样例发现需要同时考虑第一次的最大距离.

Code

D_Game on a Graph

看起来很难的图论,其实只是个博弈…
n个点需要n-1条边构成连通图,一共有m条,即可以拆去n-m+1条.
1组和2组轮流拆,直到某个人拆去一条边导致无法构成连通图则该组失败.

Code

F_Stones in the Bucket

n个堆,每个堆有 $ a_i $ 块石头,有两种操作

1 . 从某非空堆中取出一个扔掉
2 . 从某非空堆取出一个放入任意一个堆.

问要让每个堆石头数量相同,需要进行多少步.
其实就是求平均值,遍历每个堆数量大于均值则ans加上多的石头数量

Code

H_Tokens on the Segments

二维平面上每一个y=i有一条线段,区间是 [ L , R ] ,现在可以在线段任一个整数点放置标记,但是某个点放置标记后,该点所在的 Y 轴(纵向)无法再放置任何标记.求可以放置标记的最大线段数.
使用优先队列,首先对左端点进行升序排序,再依据右端点进行升序排序.
记录已经放置标记的点的x坐标,当线段左端点小于该标记时使其左端点加一.
即左端点越小优先度越高,其次右端点越小优先度越高.

Code

L_Median

有n个元素,给出m个关系,a>b,问是否有满足条件的序列,并且找到其中的中位数.
若满足条件,中位数位置输出1,其他为0,否则全输出0;
使用 Floyd 算法可以很简单的判断大小关系,若出现闭环则一定不满足.
然后统计对于第i个数,有多少个大于i的和多少个小于i的,判断中位数.

Code

M_Sekiro

签到题
给出 n 和 k ,每次操作 n 除以2并向上取整.重复 k 次.
这个题目主要在于 k 的范围在 $ 10^9 $ ,无法直接使用循环.
但是考虑的 $ 2^{32} $ 已经大于 $ 10^9 $,所以当 k 大于等于32的时候可以直接输出1,只是要记得特殊判断 n 为 0 的情况.
